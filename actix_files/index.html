Crate actix_files[−][src]

Static file serving for Actix Web.

Provides a non-blocking service for serving static files from disk.

Example

use actix_web::App;
use actix_files::Files;

let app = App::new()
    .service(Files::new("/static", ".").prefer_utf8(true));
