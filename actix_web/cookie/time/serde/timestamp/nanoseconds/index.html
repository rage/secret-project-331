Module actix_web::cookie::time::serde::timestamp::nanoseconds

source ·
Expand description

Treat an OffsetDateTime as a Unix timestamp with nanoseconds for the purposes of serde.

Use this module in combination with serde’s #[with] attribute.

When deserializing, the offset is assumed to be UTC.
