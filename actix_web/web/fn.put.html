pub fn put() -> Route

Create route with PUT method guard.

use actix_web::{web, App, HttpResponse};

let app = App::new().service(
    web::resource("/{project_id}")
        .route(web::put().to(|| HttpResponse::Ok()))
);

In the above example, one PUT route gets added:

  • /{project_id}
