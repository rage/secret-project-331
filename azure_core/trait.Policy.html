Expand description

A pipeline policy.

Policies are expected to modify the request and then call the following policy. Policies can then inspect the response, potentially signaling failure. The only runtime enforced check is that the last policy must be a Transport policy. It’s up to the implementer to call the following policy. The C generic represents the contents of the AuthorizationPolicy specific of this pipeline.
