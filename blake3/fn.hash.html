The default hash function.

For an incremental version that accepts multiple writes, see Hasher::new, Hasher::update, and Hasher::finalize. These two lines are equivalent:

let hash = blake3::hash(b"foo");

let hash = blake3::Hasher::new().update(b"foo").finalize();

For output sizes other than 32 bytes, see Hasher::finalize_xof and OutputReader.

This function is always single-threaded. For multithreading support, see Hasher::update_rayon.
