pub fn vars() -> Vars
Expand description

After loading the dotenv file, returns an iterator of (variable, value) pairs of strings, for all the environment variables of the current process.

The returned iterator contains a snapshot of the process’s environment variables at the time of this invocation, modifications to environment variables afterwards will not be reflected in the returned iterator.

Examples:


use dotenv;
use std::io;

let result: Vec<(String, String)> = dotenv::vars().collect();
