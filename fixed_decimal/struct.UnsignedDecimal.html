Struct UnsignedDecimal

Expand description

A struct containing decimal digits with efficient iteration and manipulation by magnitude (power of 10).

Supports a mantissa of non-zero digits and a number of leading and trailing zeros; used for formatting and plural selection.

ยงData Types

The following types can be converted to a UnsignedDecimal:

  • Integers, unsigned
  • Strings representing an arbitrary-precision decimal
  • Floating point values (using the ryu feature)

To create a UnsignedDecimal with fraction digits, either create it from an integer and then call UnsignedDecimal::multiplied_pow10, create it from a string, or (when the ryu feature is enabled) create it from a floating point value using [UnsignedDecimal::try_from_f64].

ยงMagnitude and Position

Each digit in a FixedDecimal is indexed by a magnitude, or the digitโs power of 10. Illustration for the number โ12.34โ:

MagnitudeDigitDescription
11Tens place
02Ones place
-13Tenths place
-24Hundredths place

Some functions deal with a position for the purpose of padding, truncating, or rounding a number. In these cases, the position sits between the corresponding magnitude of that position and the next lower significant digit. Illustration:

Position:   2   0  -2
Number:     |1|2.3|4|
Position:     1  -1

Expected output of various operations, all with input โ12.34โ:

OperationPositionExpected Result
Truncate to tens110
Truncate to tenths-112.3
Pad to ten thousands40012.34
Pad to ten thousandths-412.3400

use fixed_decimal::UnsignedDecimal;

let mut dec = UnsignedDecimal::from(250u32);
assert_eq!("250", dec.to_string());

dec.multiply_pow10(-2);
assert_eq!("2.50", dec.to_string());

pub fn concatenated_end( self, other: UnsignedDecimal, ) -> Result<Self, (UnsignedDecimal, UnsignedDecimal)>

Concatenate another UnsignedDecimal into the end of this UnsignedDecimal.

All nonzero digits in other must have lower magnitude than nonzero digits in self. If the two decimals represent overlapping ranges of magnitudes, an Err is returned, containing (self, other).

The magnitude range of self will be increased if other covers a larger range.

ยงExamples
use fixed_decimal::UnsignedDecimal;

let integer = UnsignedDecimal::from(123u32);
let fraction = UnsignedDecimal::from(456u32).multiplied_pow10(-3);

let result = integer.concatenated_end(fraction).expect("nonoverlapping");

assert_eq!("123.456", result.to_string());
