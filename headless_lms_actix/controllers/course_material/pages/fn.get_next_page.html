GET /api/v0/course-material/pages/:page_id/next-page - returns next pages info. If current page is the last page of the chapter, returns next chapters first page.

Example,

{
    "url_path": "/path-to-next/page",
    "title": "Name of the next page",
    "chapter_number": 1
}
