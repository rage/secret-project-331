Umm al-Qura Hijri Calendar (Used in Saudi Arabia)

§Era codes

This calendar supports a single era code, Anno Mundi, with code "ah"

§Month codes

This calendar is a pure lunar calendar with no leap months. It uses month codes "M01" - "M12".
