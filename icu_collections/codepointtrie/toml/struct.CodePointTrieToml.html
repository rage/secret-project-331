Struct CodePointTrieToml

pub struct CodePointTrieToml { /* private fields */ }
Expand description

A Serde-compatible struct for reading serialized CodePointTrie TOML files generated by ICU4C.

Use TryInto to convert CodePointTrieToml to a proper CodePointTrie.
