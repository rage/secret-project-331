Enum EitherProvider 

Source 
pub enum EitherProvider<P0, P1> {
    A(P0),
    B(P1),
}
Expand description

A provider that is one of two types determined at runtime.

Data provider traits implemented by both P0 and P1 are implemented on EitherProvider<P0, P1>.
