Keep the Bcc header

By default, the Bcc header is removed from the email after using it to generate the message envelope. In some cases though, like when saving the email as an .eml, or sending through some transports (like the Gmail API) that donโt take a separate envelope value, it becomes necessary to keep the Bcc header.

Calling this method overrides the default behavior.
