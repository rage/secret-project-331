Guess the MIME type of path by its extension (as defined by Path::extension()).

If path has no extension, or its extension has no known MIME type mapping, then the MIME type is assumed to be application/octet-stream.

Note

Guess is the operative word here, as there are no guarantees that the contents of the file that path points to match the MIME type associated with the pathâ€™s extension.

Take care when processing files with assumptions based on the return value of this function.

In HTTP applications, it might be preferable to not send a Content-Type header at all instead of defaulting to application/content-stream.
