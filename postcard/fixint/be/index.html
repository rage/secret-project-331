postcard::fixint

Module be 

Source
Expand description

Disables varint serialization/deserialization for the specified integer field.

Use with the #[serde(with = "postcard::fixint::be")] field attribute. The integer will always be serialized in the same way as a fixed size array, in Big Endian order on the wire.

#[derive(Serialize)]
pub struct DefinitelyBigEndian {
    #[serde(with = "postcard::fixint::be")]
    x: u16,
}
