time::serde::timestamp::microseconds

Module option 

Source
Expand description

Treat an Option<OffsetDateTime> as a Unix timestamp with microseconds for the purposes of serde.

Use this module in combination with serde’s #[with] attribute.

When deserializing, the offset is assumed to be UTC.
