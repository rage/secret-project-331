pub struct WithFilter<M, F> { /* fields omitted */ }

A MakeWriter combinator that wraps a MakeWriter with a predicate for span and event Metadata, so that the [MakeWriterExt::make_writer_for] method returns [OptionalWriter::some] when the predicate returns true, and [OptionalWriter::none] when the predicate returns false.

This is returned by the MakeWriterExt::with_filter method. See the method documentation for details.
