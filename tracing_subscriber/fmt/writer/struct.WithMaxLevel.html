Struct tracing_subscriber::fmt::writer::WithMaxLevel

pub struct WithMaxLevel<M> { /* private fields */ }
Expand description

A MakeWriter combinator that only returns an enabled writer for spans and events with metadata at or below a specified verbosity Level.

This is returned by the MakeWriterExt::with_max_level method. See the method documentation for details.

pub fn new(make: M, level: Level) -> Self

Wraps the provided MakeWriter with a maximum Level, so that it returns OptionalWriter::none for spans and events whose level is more verbose than the maximum level.

See MakeWriterExt::with_max_level for details.
