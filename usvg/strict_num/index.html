A collection of bounded numeric types.

Includes:

  • FiniteF32
  • FiniteF64
  • NonZeroPositiveF32
  • NonZeroPositiveF64
  • PositiveF32
  • PositiveF64
  • NormalizedF32
  • NormalizedF64

Unlike f32/f64, all float types implement Ord, PartialOrd and Hash, since it’s guaranteed that they all are finite.
